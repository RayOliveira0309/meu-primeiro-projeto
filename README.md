# 🧪 Simulador de Testes QA

Projeto desenvolvido para praticar conceitos de QA (Quality Assurance) utilizando JavaScript.

## 🚀 Funcionalidades
- Validação de campo de nome
- Feedback visual (sucesso, erro e alerta)
- Histórico de testes realizados
- Limpeza de dados

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 💡 Objetivo
Simular cenários de testes manuais e validação de dados, com foco em aprendizado na área de QA.
